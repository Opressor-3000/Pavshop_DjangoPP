Как делать push

1. Пишем код только находясь в своей именованной ветке 
2. выполняем git add .
3. git commit -m 'comment'
4. git push
5. после написания кода надо актуализировать данные в ветке dev
   a. переходи в ветку dev
   б. делаем git pull origin 
   в. git merge zaur
   г. git add .
   д. git commit -m 'comment'
   e. git push
6. Возвращаемся в свою именованную ветку
